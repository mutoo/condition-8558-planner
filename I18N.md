# 多语言功能使用指南 / Multi-language Feature Guide

## 中文说明

### 功能特点

1. **默认语言**：英语 (English)
2. **支持语言**：
   - 英语 (en)
   - 中文简体 (zh)

### 如何切换语言

1. 打开应用后,在页面右上角可以看到语言切换按钮
2. 点击按钮即可在英语和中文之间切换
3. 语言偏好会自动保存到浏览器本地存储 (localStorage)
4. 下次访问时会自动加载上次选择的语言

### 技术实现

- 使用 `i18next` 和 `react-i18next` 实现国际化
- 翻译文件位于 `src/i18n/locales/` 目录:
  - `en.json` - 英语翻译
  - `zh.json` - 中文翻译
- 语言配置位于 `src/i18n/config.ts`

### 如何添加新语言

1. 在 `src/i18n/locales/` 目录下创建新的 JSON 文件,如 `ja.json`
2. 复制 `en.json` 的内容并翻译为目标语言
3. 在 `src/i18n/config.ts` 中导入新的翻译文件:
   ```typescript
   import ja from './locales/ja.json'
   ```
4. 在 `resources` 对象中添加新语言:
   ```typescript
   resources: {
     en: { translation: en },
     zh: { translation: zh },
     ja: { translation: ja },  // 新增
   }
   ```
5. 更新 `LanguageSwitcher.tsx` 组件以支持多语言切换

## English Guide

### Features

1. **Default Language**: English
2. **Supported Languages**:
   - English (en)
   - Simplified Chinese (zh)

### How to Switch Language

1. After opening the app, find the language switcher button in the top-right corner
2. Click the button to toggle between English and Chinese
3. Your language preference is automatically saved to browser's localStorage
4. The app will automatically load your preferred language on next visit

### Technical Implementation

- Built with `i18next` and `react-i18next` for internationalization
- Translation files are located in `src/i18n/locales/`:
  - `en.json` - English translations
  - `zh.json` - Chinese translations
- Language configuration is in `src/i18n/config.ts`

### How to Add a New Language

1. Create a new JSON file in `src/i18n/locales/`, e.g., `ja.json`
2. Copy the content from `en.json` and translate to the target language
3. Import the new translation file in `src/i18n/config.ts`:
   ```typescript
   import ja from './locales/ja.json'
   ```
4. Add the new language to the `resources` object:
   ```typescript
   resources: {
     en: { translation: en },
     zh: { translation: zh },
     ja: { translation: ja },  // New
   }
   ```
5. Update the `LanguageSwitcher.tsx` component to support multiple language switching

## Translation Keys Structure

The translation files follow a hierarchical structure:

```json
{
  "app": {
    "title": "...",
    "description": "...",
    "footer": "..."
  },
  "info": { ... },
  "visa": { ... },
  "trip": { ... },
  "stats": { ... },
  "calendar": { ... },
  "dateModal": { ... },
  "common": { ... },
  "language": { ... }
}
```

Each section corresponds to a specific part of the application, making it easy to maintain and update translations.
